# 今天开“卷” 

![项目版本](https://img.shields.io/badge/Version-2.0%20(Feature%20Complete)-brightgreen) ![技术栈](https://img.shields.io/badge/Tech-HTML%2C%20CSS%2C%20JS-blue) ![兼容性](https://img.shields.io/badge/Responsive-PC%20%26%20Mobile-yellow)

一个纯前端、功能全面、交互体验流畅、响应式设计的安全生产知识题库Web应用。旨在提供一个零依赖、易部署、方便快捷的在线学习和检索工具。

> 🌐 **在线访问**：https://moyuhai223.github.io/QuestionBan/ （GitHub Pages，纯前端关键词搜索；拍照搜题需配服务端 OCR，否则提示手动搜索）

## ✨ 功能亮点 (Feature Highlights)

我们为这个小应用添加了许多现代化和人性化的功能：

#### 核心功能
* **题型支持**: 完美支持 **单选题、多选题、判断题** 三种类型。
* **实时模糊搜索**: 在输入框中键入任何关键词，即可在 **题干** 和 **所有选项** 中进行实时匹配，无需手动点击。
* **分类筛选**: 可在“单选”、“多选”、“判断”三种题型中自由切换，搜索范围会立即限定在当前分类下。
* **多题库切换**: 顶部下拉可在多个题库间切换，题库名取自各题库文件顶部的 `questionBankName`，下方实时显示当前题库与题数（单选/多选/判断）；错题本按题库独立记忆，刷新后保持上次选择。新增题库只需把 `database.<名>.js` 放好并在 `banks.js` 登记一行。

#### 刷题练习 & 错题本
* **练习模式**: 在关键词搜索之外提供刷题练习，支持 **题目随机**、**选项乱序**（打乱选项内容、重新编号并重算正确答案），实时统计「已答 / 正确」。
* **错题本**: 自动记录答错的题目，可 **只练错题** 集中突破，并支持一键 **清空错题本**；数据通过 `localStorage` 本地持久化，刷新或重开浏览器都不丢失。

#### 交互体验 (UX)
* **自动显示与高亮**: 搜索或切换分类后，列表中的第一个结果会自动显示详情，并在左侧列表中高亮，操作直观。
* **键盘导航**: 使用键盘的 `↑` 和 `↓` 方向键，可以流畅地在搜索结果中上下选择题目。
* **自动滚动**: 当键盘导航超出可视范围时，结果列表会自动平滑滚动，确保选中项始终可见。
* **一键清空**: 方便的“清空”按钮可以一键重置搜索框和搜索结果。
* **说明区智能隐藏**: 当用户点击分类标签开始使用后，顶部的操作说明会自动平滑隐藏，让界面更聚焦。点击“清空”后会再次出现。

#### 兼容性与性能
* **响应式设计**: 完美适配PC、平板和手机屏幕，在不同设备上均提供最佳浏览体验。
* **桌面/移动视图切换**: 在移动端设备上，提供一个特殊的切换按钮 `💻/📱`，允许用户强制切换到完整的桌面布局进行浏览。
* **性能优化**: 通过在页面加载时预先建立搜索索引，即使题库数据量巨大，搜索响应也如丝般顺滑。
* **缓存控制**: 通过 `meta` 标签和脚本版本号，强制浏览器加载最新的题库数据，避免因缓存导致内容不更新的问题。

## 📷 拍照搜题 / 截图识别（本仓库新增）

在原版"关键词搜索"之外，新增 **拍照 / 截图自动找题**：

* **一键识别**：点搜索框旁的 📷，手机直接调相机拍题，桌面则选图。
* **服务端识别**：调用**服务端 OCR**（RapidOCR / PaddleOCR ONNX，中文高精度，见下方部署）；后端不存在或调用失败时**不在浏览器本地识别**，直接提示「请手动输入关键词搜索」。
* **精确匹配**：用识别文字与**题干**做最长连续公共子串匹配，自动剔除题号、单选圈、翻页按钮等界面干扰词，直接定位到唯一题目；识别成功后自动滚动到答案区（绿色为正确项）。
* **省流**：上传前把高像素照片自适应压到长边 ≤1600 的 JPEG（体积封顶约 380KB），不传原图。

## 🖥️ 服务器部署 / 一键更新（直接从本仓库拉取）

**安装与更新同一条命令**（幂等，从 GitHub 拉最新代码 → 部署静态站 → 安装/刷新 RapidOCR 后端）：

```bash
curl -fsSL https://raw.githubusercontent.com/moyuhai223/QuestionBan/main/server/update.sh | bash
```

* 需以 root 运行（非 root 时在 `bash` 前加 `sudo`）。
* 只更新前端：在末尾加 `--static-only`（`... | bash -s -- --static-only`）。
* 脚本会把代码克隆到 `/opt/questionban-src`，静态文件部署到网站目录，OCR 后端装在 `/opt/questionban-ocr`（venv + systemd，仅监听 `127.0.0.1:1224`）。
* 网站目录默认 `/opt/1panel/www/sites/e5.zh.ci/index`，按需改 `server/update.sh` 顶部变量。

**仅首次**需在反向代理（1Panel / Nginx）里把 `/umi-ocr` 转发到 `http://127.0.0.1:1224`，之后浏览器即同源调用高精度 OCR。

> 也可不用 git、直接部署本地文件夹：`cd server && bash deploy.sh`。本地调试可 `python server/rapidocr_server.py`，或用根目录 `ocr_server.py`（把 `/umi-ocr` 同源代理到本机 Umi-OCR）。

> 题库说明：`database.js` 为当前主题库（**882 题**：单选 599 / 多选 87 / 判断 196，源自《2026 年安全生产月安全知识竞赛题库（附答案）》PDF 解析，已做答案越界/空答案/√× 校验）。备份按题数命名，需要时对调为 `database.js` 即可切换：`database.882.bak.js`（同主库）/ `database.396.bak.js`（安全知识库 396 题）/ `database.394.bak.js`（竞赛 400 道校正版 394 题）/ `database.150.bak.js`（150 单选）。

## 💾 数据管理：导入与导出

我们创建了两个强大的工具来方便您管理题库数据。

#### 1. 批量导入 (`converter.html`)
当您有大量来自PDF或Word文档的题目时：
1.  用浏览器打开 `converter.html`。
2.  将源文档的全部文本复制并粘贴到左侧的输入框。
3.  点击“开始转换”按钮。
4.  右侧会生成格式化的JS代码，全选并复制。
5.  用复制的内容完整替换 `database.js` 文件的内容并保存。

#### 2. 导出为Word文档 (`export.html`)
当您需要打印或存档题库时：
1.  用浏览器打开 `export.html` (请确保 `database.js` 文件在同级目录)。
2.  点击“生成Word格式文本”按钮。
3.  点击“复制全部内容”按钮。
4.  打开一个空白的Word文档，直接粘贴即可。我们提供了两种排版格式（答案跟随题目或答案分离），您可以根据需要修改 `export.html` 中的逻辑。

## 🔮 未来可以“卷”的方向 (Future Improvements)

随机刷题与错题本已经实现（见上方功能亮点）。在此之上还有很大的扩展空间，例如：
* **模拟考试**: 在现有随机刷题之上，增加 **计时**、整卷 **自动判分 / 及格线**、以及按题型比例 **自动组卷** 等正式考试能力。
* **用户系统**: 增加登录注册，将学习进度、错题本等数据同步到 **云端**（目前错题本仅保存在本地浏览器）。
* **题库管理后台**: 在线增删改题目、查重与答案校验，免去手动替换 `database.js`。
