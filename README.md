# 今天开“卷” - 终极版安全生产知识题库

![项目版本](https://img.shields.io/badge/Version-2.0%20(Feature%20Complete)-brightgreen) ![技术栈](https://img.shields.io/badge/Tech-HTML%2C%20CSS%2C%20JS-blue) ![兼容性](https://img.shields.io/badge/Responsive-PC%20%26%20Mobile-yellow)

一个纯前端、功能全面、交互体验流畅、响应式设计的安全生产知识题库Web应用。旨在提供一个零依赖、易部署、方便快捷的在线学习和检索工具。

## ✨ 功能亮点 (Feature Highlights)

我们为这个小应用添加了许多现代化和人性化的功能：

#### 核心功能
* **题型支持**: 完美支持 **单选题、多选题、判断题** 三种类型。
* **实时模糊搜索**: 在输入框中键入任何关键词，即可在 **题干** 和 **所有选项** 中进行实时匹配，无需手动点击。
* **分类筛选**: 可在“单选”、“多选”、“判断”三种题型中自由切换，搜索范围会立即限定在当前分类下。

#### 交互体验 (UX)
* **自动显示与高亮**: 搜索或切换分类后，列表中的第一个结果会自动显示详情，并在左侧列表中高亮，操作直观。
* **键盘导航**: 使用键盘的 `↑` 和 `↓` 方向键，可以流畅地在搜索结果中上下选择题目。
* **自动滚动**: 当键盘导航超出可视范围时，结果列表会自动平滑滚动，确保选中项始终可见。
* **一键清空**: 方便的“清空”按钮可以一键重置搜索框和搜索结果。
* **说明区智能隐藏**: 当用户点击分类标签开始使用后，顶部的操作说明会自动平滑隐藏，让界面更聚焦。点击“清空”后会再次出现。

#### 兼容性与性能
* **响应式设计**: 完美适配PC、平板和手机屏幕，在不同设备上均提供最佳浏览体验。
* **桌面/移动视图切换**: 在移动端设备上，提供一个特殊的切换按钮 `💻/📱`，允许用户强制切换到完整的桌面布局进行浏览。
* **性能优化**: 通过在页面加载时预先建立搜索索引，即使题库数据量巨大，搜索响应也如丝般顺滑。
* **缓存控制**: 通过 `meta` 标签和脚本版本号，强制浏览器加载最新的题库数据，避免因缓存导致内容不更新的问题。

## 🛠️ 技术栈 (Technology Stack)

* **HTML5**: 负责页面结构。
* **CSS3**: 使用 Flexbox 进行现代布局，并利用媒体查询 (`@media`) 实现响应式设计和视图切换。
* **JavaScript (ES6+)**: 实现所有交互逻辑，包括数据处理、DOM操作、事件监听和性能优化。
